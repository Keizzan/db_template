Project Name:
    Basic examples of CRUD 

Synopsis:
    Script contains examples of CRUD functions for simple inventory database

To run this application, you will need the following:

    Python 3.8.10,
    pip 20.0.2,
    external python3 libraries (you can install them, by running command 'pip install -r requirements.txt'): 
    mysql-connector-python==8.0.28
    protobuf==3.20.1

For Developers
-----------------
    1. Create new file and import database
    2. Load instance of one of the class related to specific table (Customer, Customer_order, Stock)
    3. Run any the avaiable CRUD functions
